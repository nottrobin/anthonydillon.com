X-FILM

X Film Ltd is a German company distributing plastic filming. They already had a German website but wanted to build custom in the UK. I was required to part design and develop this fully flash site. There were so many different products and colours that I decided early on to use XML to store the product details to speed up the development time.

This site involved:

  • Javascript
  • Flash (Actionscript 3.0)
  • Photoshop
  • PHP
  • XML
